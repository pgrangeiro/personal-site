Django - A sua cápsula de soluções web em Python

14/06/2016 | IV Encontro Carioca de Testes

O Django é um poderoso web framework em Python. Seja para soluções praticas ou sistemas robustos, vamos conhecer um pouco sobre esse mágico framework que está por trás de aplicações web como o Pinterest, Instagram, Disqus e Spotify. Vamos passear por sua arquitetura MVT e criar um CRM em 5 minutos tirando vantagens de algumas (das muitas) soluções prontas que este web framework nos disponibiliza.
